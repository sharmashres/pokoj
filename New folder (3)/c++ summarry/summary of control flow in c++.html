Summary of Control Flow in C++

  • Definition: Control flow determines the order in which statements are executed.
  • Conditional Statements: if, if-else, nested if, and switch.
  • Loops: for, while, and do-while loops for repeated execution.
  • Branching: break, continue, and goto for altering flow.
  • Switch Statement: Selects one of many code blocks to execute.
  • Loop Control: break exits a loop, continue skips an iteration.
  • Function Calls: Changes control to a called function.
  • Exception Handling: try, catch, and throw to manage errors.
