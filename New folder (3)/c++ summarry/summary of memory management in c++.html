Summary of Memory Management in C++

  • Definition: Memory management refers to allocating and deallocating memory during the execution of a program.
  • Static Memory Allocation: Done at compile time, using variables with fixed sizes.
  • Dynamic Memory Allocation: Allocated at runtime using new and delete operators.
  • new Operator: Allocates memory on the heap and returns a pointer to it.
  • delete Operator: Frees memory allocated by new.
  • Memory Leaks: Occur when allocated memory is not deallocated.
  • Smart Pointers: Unique, shared, and weak pointers to manage memory automatically.
  • Memory Pools: Pre-allocate a chunk of memory for efficiency.
