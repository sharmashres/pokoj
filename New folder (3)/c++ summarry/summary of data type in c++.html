Summary of Data Types in C++

  • Definition: Data types specify the type of data that a variable can hold.
  • Basic Data Types: int, char, float, double.
  • Derived Data Types: arrays, pointers, functions.
  • User-Defined Data Types: struct, union, enum, class.
  • Void Data Type: Represents the absence of type.
  • Boolean Data Type: bool (true or false).
  • Wide Character Type: wchar_t for extended characters.
  • Modifiers: short, long, signed, unsigned.
