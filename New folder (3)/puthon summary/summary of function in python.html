SUMMARY FOR FUNCTIONS IN PYTHON

  • Defination
    A function is a reusable block of code.
  • Define Function:
    def func_name(params):
    return value
  • Example:
    def add(a, b):
    return a + b
    print(add(3, 5)) # Output: 8
  • Arguments Types:
    Positional: func(1, 2)
    Keyword: func(a=1, b=2)
  • Return Statement:
    Returns value or None if omitted.
