SUMMARY OF DATA STRUCTURES IN PYTHON

  • ListOrdered, mutable, allows duplicates.
    Methods: append(), remove(), sort().
    Example: lst = [1, 2, 3]
  • TupleOrdered, immutable, allows duplicates.
    Faster than lists.
    Example: tpl = (1, 2, 3)
  • Set Unordered, mutable, no duplicates.
    Methods: add(), remove(), union(). Example: st = {1, 2, 3}
  • Dictionary Unordered, mutable, no duplicates.
    Unordered, key-value pairs, mutable
    Methods: keys(), values(), get().
    Example: dct = {'a': 1, 'b': 2}
