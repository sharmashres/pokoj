SUMMARY FOR LOOPS IN PYTHON

  • for Loop:Iterates over sequences.
    for i in range(5):
    print(i) # 0 1 2 3 4
  • while Loop:Runs while condition is True.
    x = 0
    while x < 5:
    print(x)
    x += 1
  • Loop Control:
    break: Exits loop early.
    continue: Skips current iteration.
    pass: Placeholder.
  • For i in range
    for i in range(5):
    if i == 3:
    break # Stops at 3
    print(i)
    else with Loops: Runs if loop completes normally.
