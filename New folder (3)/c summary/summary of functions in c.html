Summary of Functions in C

  • Definition: A function is a block of code that performs a specific task.
  • Declaration: return_type function_name(parameters);
  • Definition: return_type function_name(parameters) { // code }
  • Calling: Use function_name(arguments); to execute a function.
  • Return Statement: The return keyword is used to send a value back to the caller.
  • Types: Predefined (standard library) and User-defined functions.
  • Arguments: Pass by value and pass by reference.
  • Recursion: A function that calls itself for a repetitive task.
  • Scope: Local and global variables used within the function.
