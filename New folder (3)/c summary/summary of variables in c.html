Summary of Variables in C

  • Definition: Variables are used to store data values in memory.
  • Declaration: data_type variable_name;
  • Initialization: data_type variable_name = value;
  • Types: int, float, char, double, etc.
  • Scope: Local, Global, Static, and Automatic variables.
  • Storage Classes: auto, register, static, extern.
  • Constants: Use const keyword for constant variables.
  • Dynamic Variables: Allocated at runtime using pointers.
