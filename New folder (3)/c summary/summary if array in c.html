Summary of Arrays in C

  • Definition: Arrays store multiple values of the same type in contiguous memory locations.
  • Declaration: data_type array_name[size]; (e.g., int arr[5];)
  • Initialization: int arr[5] = {1, 2, 3, 4, 5};
  • Accessing Elements: arr[index] (zero-based indexing)
  • Looping: Use for or while loops to iterate through elements.
  • Multidimensional Arrays: int matrix[2][2] = {{1, 2}, {3, 4}};
  • Passing to Functions: Arrays are passed by reference (void func(int arr[], int size);)
  • Pointer Relationship: arr is a pointer to the first element (*arr gives the first value).
  • Dynamic Arrays: Use malloc() or calloc() for memory allocation.
